A Little Bit About Hypertext Markup Language (HTML)

•      Html is the pure backbone of the webpage

•      All of the things like the text you see like paragraphs on databases are from HTML.

•      HTML makes only the outline. There is no decoration or functions

•      In HTML, you can easily create something. All web browsers turn other things into HTML.

•      Even this presentation is running in a web browser, so it is using HTML.

•      HTML came out in 1990, but many new versions have come out throughout the years, and today we have a highly complex language called HTML5.

Cascading Style Sheets (CSS)

•      CSS is used to make the webpage or site look better and neat.

•      If you want a functioning web page and do not care about moderation, do not learn CSS.

•      CSS is also completely useless if you don’t care about how your webpage looks, but most people actually want their webpage to look nice.

Javascript (JS)

•      Javascript is a “server side” programming language. It is what makes the functions on a online calculator and other stuff.

•      The number(s) and operation(s) you input in a calculator are functioned through JS functions, and the webpage itself makes runs independently, with no need of the server.

•      JS is also really useful to change a webpage when you do something. For example, it can change the background of the page to a pale green from a deep blue.

Styling

 

Let’s make the program look good!

•         To do this, we need to create a new file.

•         So, create a new file called styles.css.

•         Preferably, save it in the directory where the original was.

•         Then, in the <head> element, add another element that says, <link>.

•         A link element is something that will let you add a stylesheet to the HTML page.

•         Well, what’s a stylesheet? A stylesheet is a fancy way to call a CSS file. A CSS file will style an element based on a label you give.

•         Your link tag should say:

<link rel=”stylesheet” href=”styles.css”></link>
